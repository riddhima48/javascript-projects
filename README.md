# ***ðŸ”¹ Color Changer / Theme Switcher***

**A simple interactive web application built using vanilla JavaScript.
Users can change the background color dynamically using buttons, generate random colors, and switch between dark and light themes.**
-------

 ### **Features**

 Change background color on button click

 Generate random colors

 Toggle between Dark Mode and Light Mode
 ---------

 ### **What I Learned**

 Using arrays to store colors

 Generating random values with Math.random()

 Manipulating styles dynamically using JavaScript

 Handling button click events
 --------

 ### **Tech Stack**

**HTML**
**CSS**
**JavaScript (Vanilla)**
Separation of Concerns: HTML for structure, CSS for styling, JS for behavior

Reusability: CSS/JS can be used across multiple HTML pages

Caching: Browsers cache external files for faster loading

Maintainability: Easier to manage and debug separate files

Collaboration: Different team members can work on different files

Organization: Cleaner, more professional project structure
